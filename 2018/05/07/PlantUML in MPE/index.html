PlantUML in MPE & Hexo

Posted on 2018-05-07

In Markdown Preview Enhanced use PlantUML syntax

Introduction

Markdown Preview Enhanced support mermaid, PlantUML, WaveDrom, GraphViz, Vega & Vega-lite, Ditaa image rendering.
You can also use Code Chunk to rendering TikZ, Python Matplotlib, Plotly etc.

Download and install Graphviz

PlantUML image preview depend on Graphviz. Download Windows version

configuration environment

Add the environment variable after install Graphviz：
Variable name：GRAPHVIZ_DOT
Variable value：C:\Program Files (x86)\Graphviz2.38\bin\dot.exe

configuration VSCode

Install PlantUML and Graphviz Preview Extensions in VSCode.
Open VSCode Setting, set

Make UML using PlantUML can use GIT for version control, and easily integrating into MarkDown, also use gravizo to render PlantUML.

Install hexo-filter-plantuml (Better than hexo-tag-plantuml)

npm install —save hexo-filter-plantuml
